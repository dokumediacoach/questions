FrageQuestion 7

Öffne die Datei questions-demo.xml in einem Texteditor deiner Wahl.

Der Anfang der Datei wird in etwa so aussehen:

<?xml version="1.0" encoding="UTF-8"?>
<questions topic="Questions Training Demo"
           language="de|en"
           xml:lang="en"
           xsi:noNamespaceSchemaLocation="xsd/questions.xsd"
           xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">

    <question nr="1">
        <intro>
            Maybe you know this situation?<br />
            In the middle of this exam preparation, someone convinced you to play around with that
            suspicious internet again and you found some useful exercise questions, but
            they look really awful.
        </intro>
        <multiple-choice>
            <option>Sorry, I don't. What was your name again?</option>
            <option>I've seen an app for this.</option>
            <option correct="true">Yes, and that really got on my nerves.</option>
            <option>What is this supposed to be? An interrogation???</option>
        </multiple-choice>
    </question>

    ...

Ja, die Datei ist wirklich schön.

  • Die XML-Datei questions-demo.xml ist die Quelle des strukturierten Inhalts für dieses Questions Training Demo.
  • Die HTML-Datei questions-demo.html ist diejenige, die du im Browser öffnest.
  • Die anderen Dateien dienen dazu, das XML in HTML umzuwandeln und dafür zu sorgen, dass das Ergebnis gut aussieht und funktioniert.

Just open the file questions-demo.xml in a text editor of your choice.

The beginning of the file will look something like this:

<?xml version="1.0" encoding="UTF-8"?>

Yes, it sure is beautiful.

  • The XML file questions-demo.xml is the source of structured content for this Questions Training Demo.
  • The HTML file questions-demo.html is the one you open in the browser.
  • The other files are for converting the XML to HTML and for the result to look and function nice.
